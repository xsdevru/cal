# 07 — Фронт на реальном бэке

> Статус: реализовано (ретроспективный issue).

## What to build

Убрать рассинхрон типов, отмеченный в `CLAUDE.md`: заменить руками-продублированные
`interface`-ы в `web/src/api.ts` на типы, сгенерированные из OpenAPI (`make types`). Сам
fetch-клиент не трогаем. Все страницы админки и публичная страница ходят на реальный бэкенд
(`:3000`), а не на Prism-мок. Компиляция и линт ловят любое расхождение с контрактом.

## Acceptance criteria

- [ ] `web/src/api.ts` использует сгенерированные типы вместо ручных `interface`-ов.
- [ ] `cd web && npm run build` (`tsc -b`) проходит.
- [ ] `npm run lint` (oxlint) без ошибок.
- [ ] Страницы (`EventTypes`, `Schedules`, `Bookings`, публичная) работают против `:3000`.
- [ ] При изменении контракта `make types` перегенерирует типы, и рассинхрон виден на компиляции.
